# 🔬 HFT Orderbook Microstructure & VPIN Orderflow Toxicity

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Real-time High-Frequency Trading (HFT) microstructure analyzer and Volume-Synchronized Probability of Toxicity (**VPIN**) calculator for Level 2/3 orderbook flow.

## ⚡ Features
- **Limit Order Book (L2/L3) Reconstruction**: Real-time bid/ask depth tracking.
- **VPIN Toxicity Metric**: Quantifies adverse selection risk before volatile price swings.

## 🚀 Quickstart
```bash
pip install -r requirements.txt
python main.py
```
